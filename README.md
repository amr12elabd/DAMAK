# DAMAK – Agricultural Development

A static website for **DAMAK**, an Egyptian agricultural development organization focused on land reclamation, irrigation, crop management, and food-security initiatives.

## About the Site

The website covers:

- **Mission & Vision** – Who DAMAK is and what it stands for
- **Key Stats** – Feddans developed, active projects, families supported
- **Services** – Land reclamation, irrigation systems, crop management, consulting, mechanization, post-harvest & export
- **Contact** – Direct contact information

## Tech Stack

Plain HTML + CSS — no frameworks, no build step, no dependencies. Fully static and deployable anywhere.

## Deployment

Hosted on **GitHub Pages** from the `main` branch, root folder.

To deploy updates:

```bash
git add .
git commit -m "Your update message"
git push origin main
```

GitHub Pages rebuilds automatically within a minute or two.
